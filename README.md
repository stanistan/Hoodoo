# Hoodoo

A Project specific todo list via the command line.

An exercise in doing things (to manage doing things) in Haskell.

Yes, this [TODO](TODO) was generated by `hoodoo`.

--

### Usage

*This needs to be run in a git project directory or subdirectory.*

`hoodoo` will create (or try to parse) a `TODO` file, which has tasks listed in the format:

```
1) [ ] do a thing!
2) [x] do another thing!
```

#### Examples

```
# adds an item to the list
hoodoo add "This needs a LICENSE"
hoodoo a "This needs a LICENSE"

# lists all items
hoodoo
hoodoo la

# list done items
hoodoo list-done
hoodoo ld

# list not completed items
hoodoo list-left
hoodoo lf

# clear out completed items
hoodoo clean
hoodoo c
```

### Build

Build it.

```
cabal build
```

Alias it.

```sh
# in your ~/.bash_profile
alias hoodoo=path/to/Hoodoo/dist/build/hoodoo/hoodoo
```
